# OfficeBT - 上班族冒险

一款以"上班族冒险"为主题的卡牌策略游戏，灵感来源于《杀戮尖塔》。

在办公室的层层楼阁中，面对刁钻的甲方、甩锅的同事、不讲理的领导，用你的职业技能和职场智慧一路向上，直到通关！

## 游戏特色

- **6 种职业角色**，各有独特玩法和专属 Buff 机制
- **回合制卡牌战斗**，每回合规划出牌策略
- **随机地图探索**，每局都是不同路线
- **卡牌奖励系统**，战斗胜利后选择新卡牌强化牌组
- **商店 / 休息 / 事件**，丰富的地图房间类型
- **自动存档**，基于 localStorage 随时继续

## 角色介绍

| 角色 | 定位 | 专属 Buff | 特色 |
|------|------|-----------|------|
| 文员 | 防御/控制 | - | 擅长流程拖延与文档攻击，生命值较高 |
| 操作工 | 纯防御/反击 | 耐心 | 化解甲方攻击、阳奉阴违，防御极高 |
| 体系 | 持续消耗 | 流程 | 低攻低防但续航极强，擅长回血抽牌 |
| 数据 | 精准核算 | 核算 | 报表分析、记账盘点，蓄力后精准打击 |
| 安全员 | 预防预警 | 预警 | 巡检预警减免伤害，隐患排查削弱敌人 |
| IT | 爆发/技术 | 加班 | 代码攻击与加班加成，能量高但血量低 |

## 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) 18+ 
- npm 9+

### 安装与运行

```bash
# 克隆仓库
git clone https://github.com/jfishpw/officeBT.git
cd officeBT

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

启动后浏览器会自动打开 `http://localhost:5173`。

### 构建发布

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 游戏玩法

### 基本操作

1. **选择角色** - 在主菜单点击开始，选择一个职业
2. **地图探索** - 点击可前往的节点移动，选择不同路线
3. **卡牌战斗** - 每回合获得能量，出牌攻击敌人或防御自身
4. **获得奖励** - 战斗胜利后选择新卡牌加入牌组
5. **击败 Boss** - 通关每层 Boss，最终到达顶层

### 战斗规则

- 每回合获得 **3 点能量**（IT 角色为 4 点）
- 出牌消耗能量，点击手牌出牌
- 攻击牌需要选择目标敌人（多敌人时）
- 点击"结束回合"进入敌人回合
- 敌人意图显示在头顶，提前规划防御
- 护甲在每回合开始时清零

### 卡牌池查看

战斗和地图界面都有"查看牌组"按钮：

- **全部** - 当前已获得的所有卡牌
- **未使用** - 本场战斗还没打出的卡牌（抽牌堆 + 手牌）
- **已使用** - 本场战斗已打出的卡牌（弃牌堆 + 消耗堆）

### 地图房间类型

| 图标 | 类型 | 说明 |
|------|------|------|
| ⚔️ | 普通战斗 | 与 1-2 个普通敌人战斗 |
| 💀 | 精英战斗 | 与强力敌人战斗，奖励更丰厚 |
| 👔 | Boss 战 | 每层最终战斗 |
| 🛒 | 商店 | 购买卡牌和移除卡牌 |
| ⛺ | 休息 | 回复生命或升级卡牌 |
| ❓ | 事件 | 随机事件，做出选择 |
| 💎 | 宝藏 | 获得奖励 |

## 技术栈

- **TypeScript** - 类型安全
- **Vite** - 快速构建与热更新
- **HTML5 DOM** - 纯 DOM 渲染，无框架依赖
- **CSS3** - Flexbox 布局 + CSS 动画
- **localStorage** - 本地存档

## 项目结构

```
src/
├── core/          # 核心模块（状态机、事件总线、随机数）
├── data/          # 数据定义（角色、卡牌、敌人、关卡）
├── game/          # 游戏逻辑（战斗、牌组、地图、存档）
├── ui/            # 界面渲染（战斗、地图、商店等场景）
├── assets/        # 静态资源
└── main.ts        # 入口文件
```
